# 一个灵活、好用、持续维护的uniapp小程序拓展

## 特性
   - 提供uniapp和uviewUi的组件提示功能
   - 鼠标悬浮在标签和属性上面会提示文档
   - 自动填充属性的默认值
   - 输入``<``弹出标签补全
   - 在标签上敲``空格``即可弹出属性提示
   - 在标签上敲``@``弹出能绑定的事件
   - 在标签上敲``:``弹出能绑定的属性值(过滤掉方法)
   - 在标签的属性上输入``空格键``弹出该属性的可选值
## 使用
   给vue文件的指定``wxml``类型,
   ```vue
   <template lang="wxml">
   </template>
   ```
## 功能预览

代码片段提供,提供组件定义查询,自动补全属性
