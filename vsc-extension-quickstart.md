# uniapp-vscode扩展

## 特性

### uniapp代码补全

- 支持小程序 API 的代码补全
- 支持 `WXML` 中组件的代码补全
- 支持 `WXML` `WXSS` `WXS` 的语法高亮
- 支持快速创建 `Page` 及 `Component`
